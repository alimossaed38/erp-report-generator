# اختبار الواجهات

1. شغّل `php tools/check_requirements.php`.
2. شغّل الخادم: `php -S 127.0.0.1:8010 -t public public/index.php`.
3. افتح الصفحات `/`, `/sales`, `/inventory`, `/finance`.
4. تحقق من البحث والفلاتر والترتيب والتنقل بين الصفحات.
5. جرّب CSV وExcel وJSON والطباعة.
6. اختبر الوضع الداكن، عرض الموبايل، والقائمة الجانبية.
7. راقب Console وNetwork وتأكد من عدم وجود أخطاء أو أصول خارجية.
