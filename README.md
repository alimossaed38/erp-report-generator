# Ali ERP Analytics

نظام تقارير وتحليلات عربي RTL مبني بـ **PHP + SQLite**، ويضم لوحة معلومات تنفيذية وتقارير للمبيعات والمخزون والمالية. النسخة الحالية أعيد تصميمها بالكامل لتكون أسرع، أوضح، وأكثر قابلية للاستخدام على الكمبيوتر والموبايل.

## أهم الإمكانيات

- لوحة معلومات تنفيذية بمؤشرات المبيعات، صافي التدفق، قيمة المخزون والتنبيهات.
- بحث وفلاتر متعددة لكل تقرير، مع اختصارات للفترات الزمنية.
- ترتيب الجداول من أي عمود وتقسيم النتائج إلى صفحات مع اختيار عدد الصفوف.
- مقارنة الفترة الحالية بالفترة السابقة وإظهار نسبة التغير.
- مؤشرات إضافية: العملاء الفريدون، الربح المتوقع من المخزون، الأصناف النافدة، هامش صافي التدفق.
- رسوم بيانية محلية بدون الاعتماد على CDN أو اتصال إنترنت.
- تصدير كل النتائج المطابقة إلى CSV وExcel، مع طباعة محسّنة أو حفظ PDF من المتصفح.
- وضع فاتح/داكن محفوظ تلقائيًا، وقائمة متجاوبة للموبايل.
- حماية أفضل للمدخلات والتصدير ورؤوس أمان HTTP.
- فهارس SQLite وتحسينات WAL وbusy timeout لسرعة واستقرار أكبر.

## المتطلبات

- PHP 8.2 أو أحدث.
- امتدادا `PDO` و`PDO_SQLite` مفعّلان.
- صلاحية كتابة على مجلد `database`.

فحص المتطلبات:

```bash
php tools/check_requirements.php
```

## التشغيل السريع

### Windows / WAMP

شغّل الملف:

```text
tools\start-windows.bat
```

أو نفّذ يدويًا:

```bash
php -S 127.0.0.1:8010 -t public public/index.php
```

ثم افتح:

```text
http://127.0.0.1:8010
```

### Linux / macOS

```bash
./tools/start.sh
```

## إعادة إنشاء البيانات التجريبية

> هذا الأمر يحذف قاعدة البيانات الحالية ويولّد بيانات تجريبية جديدة.

```bash
php database/seed.php
```

## التخصيص

عدّل `config/app.php` لتغيير:

- اسم النظام والشركة.
- اختصار العلامة.
- العملة.
- المنطقة الزمنية.
- خيارات عدد الصفوف في الجداول.

## الاختبارات

```bash
php tests/run_unit.php
```

كما يمكن فتح الصفحات الأربع والتحقق من البحث والفلاتر والتصدير والطباعة يدويًا.

## بنية المشروع

```text
app/Core          النواة والمساعدات
app/Controllers   التحكم في صفحات التقارير والتصدير
app/Models        استعلامات وتحليلات SQLite
app/Views         واجهات RTL
assets            CSS وJavaScript محليان
config            إعدادات النظام
public            نقطة تشغيل الخادم
database          المخطط والبيانات
tools            أدوات التشغيل والفحص
tests             اختبارات الوحدة
```
