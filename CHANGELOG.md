# سجل التغييرات

## غير مُصدَر

### مساعد التقارير
- إضافة مساعد محادثة عائم في كل الصفحات يفهم الطلبات العربية ويجلب التقارير الجاهزة.
- محلّل نصّي مدمج (`Intent`) بدون أي نموذج ذكاء اصطناعي خارجي أو مفتاح أو اتصال إنترنت — يبقى النظام offline.
- يغطّي التقارير الخمسة (المبيعات، العملاء، أعمار الذمم، المخزون، المالية) مع فهم الفترات الزمنية والأرقام العربية واسم العميل والحالة والنوع وأعلى-N ونيّة التصدير.
- كل رد يوضّح تفسيره ("فهمت طلبك…")، ويعيد مؤشرات + جدولًا مصغّرًا + رابط التقرير الكامل والتصدير، ويقترح أمثلة عند تعذّر الفهم.
- يعيد استخدام استعلامات الـ Repositories الموجودة دون أي SQL جديد.

## 2.0.0 — 2026-07-12

- إعادة تصميم كاملة للوحة التحكم والتقارير بهوية احترافية موحّدة.
- إضافة الوضع الداكن، التنقل المتجاوب، حالات فارغة، تنبيهات بصرية وطباعة محسنة.

### تقارير وتحليلات
- بحث متعدد، فلاتر متقدمة، اختصارات فترات، ترتيب أعمدة وتقسيم صفحات.

### تصدير وأداء وأمان
