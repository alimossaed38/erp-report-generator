# سجل التغييرات

## 2.0.0 — 2026-07-12

### تصميم وتجربة استخدام
- إعادة تصميم كاملة للوحة التحكم والتقارير بهوية احترافية موحّدة.
- إضافة الوضع الداكن، التنقل المتجاوب، حالات فارغة، تنبيهات بصرية وطباعة محسنة.
- إزالة الاعتماد على Google Fonts وChart.js الخارجي؛ جميع الأصول تعمل محليًا.

### تقارير وتحليلات
- بحث متعدد، فلاتر متقدمة، اختصارات فترات، ترتيب أعمدة وتقسيم صفحات.
- مقارنة الفترة الحالية بالسابقة ونسب اتجاه للمؤشرات.
- مؤشرات موسعة للمبيعات والمخزون والمالية.
- قوائم أفضل المنتجات، المصروفات حسب التصنيف، والمخزون الحرج.

### تصدير وأداء وأمان
- تصدير النتائج إلى CSV وExcel.
- تصدير جميع السجلات المطابقة للفلاتر بدل عدد محدود.
- حماية خلايا الجداول من Spreadsheet Formula Injection.
- إضافة فهارس SQLite وWAL وbusy timeout.
- التحقق من التواريخ والقيم المسموحة والمدخلات الرقمية.
- إضافة رؤوس أمان HTTP ومعالجة أخطاء مركزية.

### تشغيل وصيانة
- إعداد مركزي في `config/app.php`.
- أداة فحص متطلبات وتشغيل سريع لـ Windows وLinux/macOS.
- تحديث الاختبارات لتعمل بمسار PHP الحالي بدل مسار WAMP ثابت.
